2.
	== convertar types áður en það ber saman, þannig ef maðurreynir t.d. að bera saman true eða false með tölu þá convertast true eða false í 1 eða 0.
	
		console.log(99 == "99"); // true
		console.log(0 == false); // true
	
	
	=== convertar ekki, þá verða hlutirnir að vera eins

		console.log(99 === "99"); // false
		console.log(0 === false); // false
